# WeMD 内置主题

脚本 `--theme` 使用下表 `id`。

| id                 | 名称       | 适用场景                       |
| ------------------ | ---------- | ------------------------------ |
| default            | 默认主题   | 通用、轻量                     |
| modern-editorial   | 编辑部手记 | 高信息密度长文（**默认推荐**） |
| clear-guide        | 清晰指南   | 教程、操作手册                 |
| knowledge-base     | 知识库     | 文档、Wiki 风格                |
| academic-paper     | 学术论文   | 引用、脚注、学术排版           |
| data-blueprint     | 数据蓝图   | 商业分析、数据复盘             |
| eastern-notes      | 东方笺谱   | 人文长文                       |
| whitespace-gallery | 留白画册   | 图片故事、人物专访             |
| luxury-gold        | 黑金奢华   | 品牌、高端选题                 |
| morandi-forest     | 莫兰迪森林 | 生活方式、文艺                 |
| sunset-film        | 落日胶片   | 摄影、旅行、情绪向             |
| receipt            | 购物小票   | 趣味、清单体                   |
| aurora-glass       | 极光玻璃   | 特殊风格                       |
| bauhaus            | 包豪斯     | 特殊风格                       |
| cyberpunk-neon     | 赛博朋克   | 特殊风格                       |
| neo-brutalism      | 新粗野主义 | 特殊风格                       |
| template           | 主题模板   | 开发用                         |
